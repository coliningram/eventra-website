Tier-Toggle Mock

Internal design mock — three tier-toggle UI proposals for /journeys/ detail pages. Pick one; sub-task C builds the chosen variant on the Rwanda detail page.

Each variant is rendered with both a 3-tier (Cape: Classic / Signature / Prestige) and 2-tier (Rwanda: 4★ / 5★) configuration, so you can see how each pattern handles both row counts.
